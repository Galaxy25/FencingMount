# Overview

A updated school project made with Fusion 360. It was designed to mount a fencing sabre for practice without the need of screws
![IMG_0716](https://user-images.githubusercontent.com/84703098/186964858-950f6f4d-9091-481b-9cd9-4ae4b0c892c0.jpg)

## Installation

1. Superglue the hand and mount together.

    ![IMG_0718](https://user-images.githubusercontent.com/84703098/186833219-007a0d03-9335-47e2-8bd4-159db9975f83.jpg)

2. Slip the clamp between the base and table brim. I recommend placing a towel between them to avoid scraping

    ![IMG_0720](https://user-images.githubusercontent.com/84703098/186833257-3b8aa96a-ac90-4c11-b4d2-6662a044aa69.jpg)

3. Unscrew blade and push it through the holes.

    ![IMG_0719](https://user-images.githubusercontent.com/84703098/186833282-77193c37-161a-406f-b479-577bf7cb187e.jpg)

4. Wrap the guard back onto the bolt and secure it firmly.

    ![IMG_0717](https://user-images.githubusercontent.com/84703098/186833455-19fa16e7-ec8e-4ef2-b1b1-49b9731b6098.jpg)
